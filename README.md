# 多段集結 時刻計算ツール

モバイル戦略ゲームの「多段集結」において、複数プレイヤーが指定時刻に同時着弾するための出兵時刻を計算する Web ツール。

**URL:** https://web-ks-time-calc.vercel.app

---

## 機能

- ターゲット時刻（UTC hhmmss 形式）の入力
- 参加者ごとの行軍時間（秒）を登録
- 3パターンの出兵時刻を一括計算
- 結果をゲームチャット用にワンクリックコピー
- スマホ対応

---

## 計算パターン

### 弾着時刻合わせ（緑）

全員がターゲット時刻 **T** に着弾する。

```
出兵時刻 = T − 自分の行軍時間
```

### 出発時刻合わせ・最大基準（青）

最長行軍者が T に出発し、全員が同時刻に着弾する。

```
出兵時刻 = T + (max_march − 自分の行軍時間)
着弾時刻 = T + max_march
```

### 出発時刻合わせ・最小基準（黄）

最短行軍者が T に出発し、全員が同時刻に着弾する。行軍時間が未入力の参加者は対象外。

```
出兵時刻 = T − (自分の行軍時間 − min_march)
着弾時刻 = T + min_march
```

## 開発

### 構成

```
web-ks-time-calc/
├── index.html    メインページ
├── style.css     スタイル
├── app.js        計算ロジック・UI制御
└── vercel.json   Vercel デプロイ設定
```

フレームワーク不要。HTML + CSS + JavaScript のみで動作する。

### ローカル確認

```bash
# index.html をブラウザで開くだけで動作する
open index.html
```

### デプロイ

```bash
vercel --prod
```
