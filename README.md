# 多段集結 時刻計算ツール

モバイル戦略ゲームの「多段集結」において、複数プレイヤーが指定時刻に同時着弾するための出兵時刻を計算する Web ツール。

**URL:** https://web-ks-time-calc.vercel.app

---

## 機能

### 自陣計算

- ターゲット時刻（UTC hhmmss 形式）の入力
- 参加者ごとの行軍時間（秒）を登録
- 3パターンの出兵時刻を一括計算
- 結果をゲームチャット用にワンクリックコピー
- Memory スロット（5件）による複数タブ対応
- Memory のエクスポート・インポート・リセット

### 相手の着弾時刻を計算

計算元UTC・集結時間・行軍時間から相手の着弾時刻を算出。算出結果をターゲット時刻にワンクリック転記可能。

```
着弾時刻 = 計算元UTC + 集結時間 + 行軍時間
```

### 相手の行軍時間を算出

ゲーム画面から読み取れる情報（UTC・集結時間・行軍時間表示値）を組み合わせて、相手の実際の行軍時間を逆算する。

```
行軍開始時刻 = UTC（集結時間取得時刻） + 集結時間
到着時刻     = UTC（行軍時間取得時刻） + 行軍時間（ゲーム表示）
行軍時間     = 到着時刻 − 行軍開始時刻
```

算出結果を相手着弾計算の行軍時間欄にワンクリック転記可能。

---

## 計算パターン（自陣）

### 弾着時刻合わせ（緑）

全員がターゲット時刻 **T** に着弾する。

```
出兵時刻 = T − 自分の行軍時間
```

### 出発時刻合わせ・最大基準（青）

最長行軍者が T に出発し、全員が同時刻に着弾する。

```
出兵時刻 = T + (max_march − 自分の行軍時間)
着弾時刻 = T + max_march
```

### 出発時刻合わせ・最小基準（黄）

最短行軍者が T に出発し、全員が同時刻に着弾する。行軍時間が未入力の参加者は対象外。

```
出兵時刻 = T − (自分の行軍時間 − min_march)
着弾時刻 = T + min_march
```

## 開発

### 構成

```
web-ks-time-calc/
├── index.html    メインページ
├── style.css     スタイル
├── app.js        計算ロジック・UI制御
├── help.js       ヘルプモーダルのコンテンツ
└── vercel.json   Vercel デプロイ設定
```

フレームワーク不要。HTML + CSS + JavaScript のみで動作する。

### ローカル確認

```bash
# index.html をブラウザで開くだけで動作する
open index.html
```

### デプロイ

```bash
vercel --prod
```
